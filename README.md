# [Puzzler](https://kuzzler.netlify.app/)
Images To Puzzle

## Features
- Random Images On refresh
- Search for Images and get their puzzles
- Convert Image from Url to a Puzzle
- Upload Local Image

## APIs Used
 - [Picsum](https://www.picsum.photos)
 - [Loremflickr](https://loremflickr.com)
